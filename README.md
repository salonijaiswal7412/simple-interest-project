# Simple Interest Project

A small Bash script to calculate simple interest.

## Files
- LICENSE (Apache 2.0)
- README.md
- CODE_OF_CONDUCT.md
- CONTRIBUTING.md
- simple-interest.sh

## Usage
chmod +x simple-interest.sh
./simple-interest.sh <principal> <rate> <time>

Example:
./simple-interest.sh 1000 5 2
# Output: Simple Interest = 100.00
